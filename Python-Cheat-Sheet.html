Regular expressions

module for regex: import re

r"regExp" string definition with convention where back-slashes need only be escaped once, e.g.: r"\s*" is fine

 \d Matches any decimal digit; equivalent to the set [0-9]

 \D The complement of \d. It matches any non-digit character; equivalent to the set [^0-9]

 \s Matches any whitespace character; equivalent to [ \t\n\r\f\v]

 \S The complement of \s. It matches any non-whitespace character; equiv. to [^ \t\n\r\f\v]

 \w Matches any alphanumeric character; equivalent to [a-zA-Z0-9_]

    With LOCALE, it will match the set [a-zA-Z0-9_] plus characters defined as letters for the current locale

 \W Matches the complement of \w

 \b Matches the empty string, but only at the start or end of a word

 \B Matches the empty string, but not at the start or end of a word

groups caught by parenthesis can be back referenced with \n (n being the one-based group number)

re.search(r"...", str[, pos[, endpos]][, <flags>]) -> None or match object (anywhere in str)

re.match(r"...", str[, pos[, endpos]][, <flags>] ) -> None or match object (only from beginning of str)

 match object has methods: group([n]) matching string, span([n]) a tuple with start and end positions

  start([n]), end([n]) deconstructs this tuple into its two values, positions are zero based and end is exclusive

  parameter n which is one-based gives same with regex groups (what is caught between parenthesis). n=0 means the same than no n at all

  multiple parameters n can be provided in case tuples are returned (e.g. mo.group(1, 3) -> (mo.group(1), mo.group(3))

  groups() provides a tuple of all groups

 named groups and back references

findall(r"...", string[, <flags>]) -> list of matching substrings

split(r"...", string[, <flags>]) -> list of substrings between regex matches as separators

sub(r"...", replacementStrOfFct, string, max=0) -> return string with substitution - subn same as sub, returns a tuple (replacedString, numberOfReplacementsMade) 

 max: max number of replacements to make (0: unbounded)

 in case of function as replacement, this function takes a match object as parameters and returns a string

  re.I re.IGNORECASE - re.L re.LOCALE

set, frozenset

mutable and immutable unordered collection of distinct objects, supports: x [not] in s, len(s), for x in s

constructors: set([iterable]), frozenset([iterable])

s.isdisjoint(other) do two sets have no element in common?

s.issubset(other), s <= other is every element in s is in other?

s < other is s a proper subset of other, that is, s <= other and s != other?

s.issuperset(other), s >= other is every element in other is in s?

s > other is s a proper superset of other, that is, s >= other and s != other?

s.union(*others), s | other | ... a new set with elements from the s and all others

s.intersection(*others), s & other & ... a new set with elements common to the s and all others

s.difference(*others), s - other - ... a new set with elements in s that are not in the others

s.symmetric_difference(other), s ^ other a new set with elements in either s or other but not both

s.copy() a new set with a shallow copy

operations only possible on set (as frozenset is immutable):

  s |= other | ...

  s &= other & ...

  s.difference_update(other), s -= other | ...

  s.symmetric_difference_update(other), s ^= other

  s.add(elem), s.remove(elm), s.discard(elm) - remove raises KeyError if elm is not in set

  s.pop() removes arbitrary element, raises KeyError if set is empty

  s.clear()

decorators

class entryExitAsObj: # decorator as class

  def __init__(self, f):

    self.f = f

  def __call__(self, *args):

    print(self.f.__name__ + "@")

    self.f(*args)

    print("@" + self.f.__name__)

def func0(*args):

  from inspect import stack # this will allow us to get caller's function name

  print("In " + stack()[1][3] + " with " + ", ".join(list(args)))

@entryExitAsObj # use of decorator on function

def func1(x):

  func0(x)  

@entryExitAsObj

def func2(x, y):

  func0(x, y)

func1("hello world") # func1@ / In func1 with hello world / @func1

func2("hello", "world") # func2@ / In func2 with hello, world / @func2

def entryExitAsFct(fct): # decorator as function

  def newFct(*args): # closure

    args2=()

    for item in args:

      args2 = args2 + (item.upper(),)

    print("@>" + fct.__name__)

    fct(*args2)

    print("@<" + fct.__name__)

  newFct.__name__ = fct.__name__

  return newFct

@entryExitAsFct

def func3(x):

  func0(x)  

@entryExitAsFct

def func4(x, y):

  func0(x, y)  

func3("hello world") # @>func3 / In func3 with HELLO WORLD / @<func3

func4("hello", "world") # @>func4 / In func4 with HELLO, WORLD / @<func4

modules and packages

A module is simply a Python file <mod>.py with functions, variables, possibly code to execute

How to import it:

 import <module1>[, <module2>[, ...<moduleN>]...]

  in this case, entities from moduleN should be called with a prefix: <moduleN>.<entity>

 from <module> import <entity1>[, <entity2>...]

  in this case, no need to prefix entityN: it is directly accessible

 from <module> import *

  this way all entities are imported from module without any need to prefix them

Locating modules:

 in current directory, in environment variable PYTHONPATH, and paths where Python installed libraries

 can be seen in sys.path (sys being of course a module) sys.path.append(<filePathStr>) can be used to dynamically enrich module locations 

Functions

 dir(<module>) returns a list of variables and functions a module contains

 reload(<module>) re-execute the top-level code of a module (which is normally run on the first import of a module)

Package

 under a <folder>, Python modules are copied plus a file __init__.py which imports all those modules which allows to only write import <folder>

Privacy considerations:

 variables with one underscore are considered private and are not exported with 'from <module> import *'

 members of classes with double underscore __var are not (easily) accessible because they are mangled this way _<class>__<var>
